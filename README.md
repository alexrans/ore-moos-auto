# ore-moos-auto
ore-moos-auto 自动化远程部署Ubuntu挖矿环境，自主选择创建钱包数据，设置Gas费用，设置Rpc，工作室必备工具。

使用方法：
 python ore-moos-auto.py
填写VPS SSH信息  IP 用户（建议root） 密码  端口
等待环境部署完毕 开始创建钱包  输入您需要创建并多开的钱包数量

部署完毕：
自行SSH登录到部署完毕的VPS
运行./run.sh
进行相关操作
